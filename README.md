
## Backend 
```shell
cd server
npm i  
npm start 
```

## Client 
```bash 
npm i  
npm start 
```

## Docker 
```shell
docker-compose --env-file .env up -d --build
# some adds 
docker-compose logs
docker-compose down 
```

Then go to `localhost:3000`
